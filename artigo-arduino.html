ARDUINO:
A PONTE ENTRE A LÓGICA E A ELETRÔNICA

O Arduino é o cérebro dos nossos projetos na GAIA Robótica. Usamos esta plataforma para controlar desde os motores do Robô Omnidirecional até a telemetria básica dos nossos drones em Uberaba.

O Cérebro: ATMega328P

Este microcontrolador de 8 bits permite a execução de instruções em um único ciclo de clock, abstraindo funções complexas em código legível.

void setup() { pinMode(13, OUTPUT); } void loop() { digitalWrite(13, HIGH); delay(1000); }
