ARDUINO:
O QUE É E
COMO FUNCIONA?

A plaquinha que move nossos robôs. Entenda o que é um microcontrolador, como programar do zero e por que o Arduino é a porta de entrada para a robótica.

Compartilhe: 📸 Instagram 💬 WhatsApp 💼 LinkedIn

O Arduino é uma das ferramentas mais importantes da robótica moderna. Na GAIA, usamos Arduino no Robô Corredor, no Robô Omnidirecional, nas oficinas educacionais e como base para aprender eletrônica. Mas o que é, exatamente?

💡 Definição simples: Arduino é uma placa eletrônica com um microcontrolador que você programa para controlar componentes físicos — LEDs, motores, sensores, buzzers e muito mais.

O que tem dentro de um Arduino?

A placa mais famosa, o Arduino Uno, parece simples — mas tem tudo que você precisa para começar:

🧠
Microcontrolador

O ATmega328P — o "cérebro" da placa. Executa seu código linha por linha.

⚡
Pinos Digitais

14 pinos que podem ligar (HIGH) ou desligar (LOW) — para controlar LEDs, relés, motores.

📡
Pinos Analógicos

6 pinos que leem valores variáveis — temperatura, luz, distância via sensores.

🔌
Porta USB

Para conectar no computador, fazer upload do código e alimentar a placa.

🕐
Clock 16 MHz

Velocidade de processamento. Executa 16 milhões de instruções por segundo.

💾
Memória Flash

32KB onde seu código fica salvo mesmo sem energia.

Seu primeiro código

Todo mundo começa com o mesmo programa: piscar um LED. É o "Olá Mundo" da eletrônica. Veja como é simples:

// Pisca o LED embutido no pino 13 void setup() { pinMode(13, OUTPUT); // define pino 13 como saída } void loop() { digitalWrite(13, HIGH); // liga o LED delay(1000); // espera 1 segundo digitalWrite(13, LOW); // desliga o LED delay(1000); // espera 1 segundo }

Todo programa Arduino tem duas funções principais: setup() — que roda uma vez ao ligar — e loop() — que fica repetindo para sempre. É o ciclo básico de qualquer robô.

Como a GAIA usa Arduino

No Robô Corredor, usamos Arduino para ler a bússola e controlar os motores de forma autônoma. O robô segue uma direção específica sem ninguém no controle — tudo programado em C++ rodando no microcontrolador.

🤖 Na prática: o Arduino lê o sensor de bússola, compara o ângulo atual com o ângulo desejado, e ajusta os motores automaticamente para manter a trajetória. Isso é robótica real — e você aprende isso nas nossas oficinas!

Arduino vs ESP32 — qual a diferença?

Nos projetos mais avançados, usamos também o ESP32 — um microcontrolador mais poderoso com Wi-Fi e Bluetooth integrados. É o que usamos no Veículo Autônomo.

Para quem está começando, o Arduino Uno é perfeito. Para projetos com comunicação sem fio ou mais processamento, o ESP32 é a escolha.
