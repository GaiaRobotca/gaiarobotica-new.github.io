01 — Definição

IA não é mágica, é estatística aplicada

Na engenharia, Inteligência Artificial é o campo que estuda sistemas capazes de tomar decisões ou fazer previsões a partir de dados — sem que cada regra tenha sido escrita manualmente por um programador. É a diferença entre programar um resultado e programar a capacidade de aprender um resultado.

No GAIA Robótica, usamos IA como ferramenta prática: para que um robô encontre padrões em dados sensoriais que seriam impossíveis de descrever com regras fixas — como reconhecer o formato de um obstáculo ou estabilizar um drone em vento variável.

Programação tradicional x IA

Na programação tradicional, você escreve: "se sensor > X, vire à esquerda". Em IA, você mostra ao sistema milhares de exemplos de quando virar à esquerda, e ele aprende sozinho o padrão que liga sensor a decisão.

02 — Categorias

Os três níveis de "inteligência"

O termo IA cobre técnicas muito diferentes entre si. Entender a hierarquia evita confusão — nem todo projeto precisa de deep learning.

Baseada em Regras

Lógica condicional explícita, escrita por humanos. Rápida e previsível, mas não "aprende". Exemplo: controle Bang-Bang de um seguidor de linha.

Machine Learning

O sistema aprende padrões a partir de dados de treino, sem que as regras sejam escritas explicitamente. Exemplo: classificar imagens de objetos.

Deep Learning

Subcampo do ML que usa redes neurais com múltiplas camadas para aprender representações complexas. Exemplo: visão computacional avançada.

03 — Funcionamento

Como uma máquina "aprende"

O processo de Machine Learning segue, de forma simplificada, três etapas fundamentais que se repetem até o modelo atingir uma precisão aceitável.

1
Coleta de dados (Dataset)
Milhares de exemplos rotulados — fotos de objetos com seus nomes, leituras de sensores com o resultado esperado. Quanto mais representativo o dataset, melhor o modelo generaliza.
2
Treinamento
O algoritmo ajusta parâmetros internos repetidamente, comparando suas previsões com o resultado real e corrigindo o erro — um processo matemático chamado otimização.
3
Inferência
Com o modelo treinado, ele passa a receber dados novos (nunca vistos) e produzir previsões em tempo real — é essa etapa que roda embarcada no robô.
Ponto importante

Treinamento é pesado computacionalmente e geralmente feito em computador potente. Inferência precisa ser rápida e leve — é o que roda no Arduino, ESP32 ou Raspberry Pi embarcado no robô.

04 — Aplicação no GAIA

Onde a IA aparece nos nossos projetos

No GAIA Eletroquad, a IA aparece principalmente em duas frentes: visão computacional, para o drone identificar marcadores e obstáculos durante o voo autônomo; e controle adaptativo, onde algoritmos ajustam parâmetros de voo em tempo real conforme as condições mudam.

Vale destacar que nem todo comportamento "inteligente" de um robô usa Machine Learning — muitas vezes um bom controlador PID bem ajustado (veja o artigo dedicado) resolve o problema com muito menos complexidade e mais previsibilidade, o que é essencial em competições.

05 — Glossário

Termos que você vai encontrar

Dataset
Conjunto de dados usados para treinar ou testar um modelo.
Overfitting
Quando o modelo "memoriza" os dados de treino em vez de aprender o padrão geral, falhando em dados novos.
Rede Neural
Estrutura matemática inspirada em neurônios biológicos, organizada em camadas que processam informação progressivamente.
Inferência
Etapa em que o modelo já treinado é usado para gerar previsões sobre dados novos, em produção.
